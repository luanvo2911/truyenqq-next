API
Main api:
1. https://api.mangadex.org
  - /manga: list all manga
  Ex: https://api.mangadex.org/manga?title=I Thought She Was a Yandere, but Apparently She's Even Worse (search base on title)

  - https://api.mangadex.org/manga?contentRating[]=suggestive: get suggestive manga list

  - /cover/:id : get all covers of a manga / id is cover_art in relationship field
  Ex: https://api.mangadex.org/cover/9b3d511f-41b0-48d5-92da-50707b2f58a0

2. https://uploads.mangadex.org
  - https://uploads.mangadex.org/covers/:manga-id/:cover-filename
  Ex: https://uploads.mangadex.org/covers/66b0511a-528c-43de-b572-a7ff7a30ebd9/4f5ebead-fa60-449d-9c6c-e0034e5b1df7.png
